# Portfolio Kariny - Edição de Vídeos


## 🎬 Serviços

- YouTube Shorts
- TikTok
- Gamers & Streamers
- Edição de Fotos
- Conteúdos com IA

## 🚀 Tecnologias

- React 18
- Vite
- CSS3 (Gradientes e Animações)
- Arquitetura MVC

## 📦 Instalação

```bash
npm install
```

## 🔧 Desenvolvimento

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```
